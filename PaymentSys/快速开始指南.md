# 快速开始指南

## 1. 准备工作

### 1.1 环境要求
- ✅ JDK 8 或更高版本
- ✅ Oracle 19c 数据库
- ✅ Maven 3.6+
- ✅ IDE（推荐 IntelliJ IDEA）

### 1.2 获取渣打银行API凭证
联系渣打银行获取：
- API Key（商户ID）
- API Secret（签名密钥）
- API Endpoint（API端点）

## 2. 数据库初始化

### 2.1 创建数据库用户
```sql
-- 以DBA身份连接
CREATE USER payment_user IDENTIFIED BY payment_pass;
GRANT CONNECT, RESOURCE TO payment_user;
GRANT CREATE VIEW, CREATE SEQUENCE TO payment_user;
```

### 2.2 执行初始化脚本
```bash
cd C:\WorkSoftware\a_program\selft\smartoneCloud\PaymentSys
sqlplus payment_user/payment_pass@localhost:1521/ORCL @src/main/resources/db/schema.sql
```

### 2.3 验证表创建
```sql
SELECT table_name FROM user_tables 
WHERE table_name LIKE 'PAYMENT%';

-- 应该看到5张表：
-- PAYMENT_TRANSACTION
-- PAYMENT_CALLBACK_LOG
-- PAYMENT_RECONCILIATION
-- PAYMENT_ORDER_MAPPING
-- PAYMENT_CONFIG
```

## 3. 配置应用

### 3.1 编辑配置文件
编辑 `src/main/resources/application.yml`：

```yaml
spring:
  datasource:
    url: jdbc:oracle:thin:@localhost:1521:ORCL
    username: payment_user
    password: payment_pass

scb:
  api:
    endpoint: https://api.standardchartered.com/payment
    key: YOUR_API_KEY_HERE
    secret: YOUR_API_SECRET_HERE

payment:
  callback:
    url: http://your-domain.com:8080
```

### 3.2 或使用环境变量（推荐生产环境）
```bash
set DB_USERNAME=payment_user
set DB_PASSWORD=payment_pass
set SCB_API_KEY=your_api_key
set SCB_API_SECRET=your_api_secret
```

## 10. 常见问题

### 问题1：数据库连接失败
**症状**: `java.sql.SQLException: IO Error: The Network Adapter could not establish the connection`

**解决**:
1. 检查Oracle服务是否启动
2. 检查数据库连接信息是否正确
3. 检查防火墙设置

### 问题2：乐观锁冲突
**症状**: 并发请求时部分失败

**解决**: 这是正常的，系统会自动重试

### 问题3：回调签名验证失败
**症状**: 回调日志中 `SIGNATURE_VALID = 0`

**解决**:
1. 检查API Secret是否正确
2. 确认签名算法与银行一致（HMAC-SHA256）

### 问题4：对账任务不执行
**症状**: 没有对账记录生成

**解决**:
1. 检查 `@EnableScheduling` 是否添加
2. 查看日志是否有异常
3. 确认cron表达式正确
