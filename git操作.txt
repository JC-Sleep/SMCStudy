初始化本地仓库
git init

执行成功后，你会看到提示：Initialized empty Git repository in ...，并且文件夹里会多出一个隐藏的 .git 文件夹。


# 查看当前远程地址
git remote -v

# 移除旧的 (如果是 GitLab)
git remote remove origin

# 添加新的 GitHub B 地址 (使用 SSH 格式)
git remote add origin https://github.com/JC-Sleep/SMCStudy.git

# 添加所有文件
git add .

# 提交更改
git commit -m "Initial commit"

# 推送代码
git push -u origin main


1. 查看当前工作区（未提交）的修改数量
方法一：最精确统计（推荐）
使用 git diff 配合统计参数，可以直观看到有多少个文件被修改，以及增加了多少行代码。
git diff --stat

方法二：仅查看文件列表
如果你只想看文件名，不想看修改了多少行：
git diff --name-only

方法三：快速计数（纯数字）
如果你只想要一个数字（例如在脚本中使用），可以结合 wc -l（行计数命令）：
git diff --name-only | wc -l

2. 查看已暂存（已 add）的修改数量
如果你已经执行了 git add .，想看暂存区里有多少修改：
git diff --staged --stat

3. 查看最近一次提交（HEAD）的修改数量
如果你想回顾刚才那次提交改了多少文件：
git show --stat

或者查看更详细的统计信息：
git show --shortstat

4. 查看两个分支之间的差异
比如你想看当前分支比 main 分支多改了多少文件：
git diff main..HEAD --stat
